# 🍕 Pizza Sales Data Analysis (SQL + Power BI)

## 📌 Project Overview

This project presents an end-to-end data analysis of a pizza sales dataset to uncover actionable business insights. The analysis focuses on revenue performance, customer ordering behavior, and product-level trends using SQL for data processing and Power BI for visualization.

---

## 🎯 Objective

The primary goal of this project is to transform raw transactional data into meaningful insights that can help a business:

* Improve revenue generation
* Optimize product offerings
* Understand customer purchase patterns
* Support data-driven decision-making

---

## 🛠️ Tools & Technologies

* **SQL (PostgreSQL)** → Data cleaning, transformation, and KPI calculations
* **Power BI** → Interactive dashboard and data visualization
* **Excel/CSV** → Data source and initial inspection

---

## 🧹 Data Preparation & Cleaning

The dataset was structured and cleaned using SQL to ensure accuracy and consistency:

* Verified and enforced appropriate data types
* Checked for and handled missing/null values
* Removed inconsistencies and duplicate records
* Created calculated metrics such as total revenue and order-level KPIs

---

## 📊 Key Performance Indicators (KPIs)

The following KPIs were developed using SQL:

* **Total Revenue**
* **Average Order Value (AOV)**
* **Total Orders**
* **Total Pizzas Sold**
* **Average Pizzas per Order**

---

## 📈 Analysis & Insights

### 🔹 Sales Performance

* Total revenue provides a clear measure of overall business performance
* A small group of top-selling pizzas contributes a significant portion of revenue

### 🔹 Product Insights

* Certain pizza categories dominate both sales volume and revenue
* Large-sized pizzas generate higher revenue contribution compared to other sizes

### 🔹 Customer Behavior

* Average order value indicates customer spending patterns
* Customers tend to order multiple pizzas per transaction

### 🔹 Time-Based Trends

* Daily trend analysis reveals peak ordering days
* Monthly trends highlight seasonal or demand fluctuations

---

## 📊 Dashboard Features

The Power BI dashboard includes:

* KPI cards for quick business overview
* Sales trends (daily & monthly)
* Category-wise and size-wise revenue distribution
* Top 5 and bottom 5 performing products
* Interactive filters for dynamic analysis
